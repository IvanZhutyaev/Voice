# Руководство по вкладу в проект

Спасибо за интерес к проекту Глас! Мы приветствуем вклад от сообщества.

## Как внести вклад

1. Форкните репозиторий
2. Создайте ветку для вашей фичи (`git checkout -b feature/AmazingFeature`)
3. Закоммитьте изменения (`git commit -m 'Add some AmazingFeature'`)
4. Запушьте в ветку (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## Стандарты кода

- Следуйте PEP 8 для Python кода
- Используйте TypeScript для frontend
- Пишите понятные комментарии
- Добавляйте тесты для новой функциональности

## Процесс разработки

1. Обсудите большие изменения в Issue перед началом работы
2. Убедитесь, что все тесты проходят
3. Обновите документацию при необходимости
4. Убедитесь, что код соответствует стандартам проекта

## Вопросы?

Создайте Issue для обсуждения.
